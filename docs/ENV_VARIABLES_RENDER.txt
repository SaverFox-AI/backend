=== ENVIRONMENT VARIABLES FOR RENDER ===

=== AI PYTHON SERVICE (saverfox-ai-python) ===

Required:
LLM_PROVIDER=openai
OPENAI_API_KEY=sk-...your-key...
OPIK_API_KEY=...your-opik-key...

Optional (with defaults):
OPENAI_MODEL=gpt-4
OPENAI_TEMPERATURE=0.7
OPENAI_MAX_TOKENS=1000
OPIK_WORKSPACE=default
OPIK_PROJECT_NAME=saverfox-ai
DEBUG=false
API_PREFIX=/api
CORS_ORIGINS=["*"]
CORS_ALLOW_CREDENTIALS=true

=== API TYPESCRIPT SERVICE (saverfox-api-ts) ===

Required:
# Database (copy from PostgreSQL service)
DB_HOST=dpg-xxxxx.singapore-postgres.render.com
DB_PORT=5432
DB_USERNAME=saverfox
DB_PASSWORD=...from-database...
DB_DATABASE=saverfox_db

# JWT Secret (generate random 32+ chars)
JWT_SECRET=your-super-secret-jwt-key-min-32-characters-long

# AI Service URL (copy from AI service)
AI_SERVICE_URL=https://saverfox-ai-python.onrender.com

Optional (with defaults):
NODE_ENV=production
API_PREFIX=api
JWT_EXPIRATION=24h
AI_SERVICE_TIMEOUT=30000
AI_SERVICE_MAX_RETRIES=3
AI_SERVICE_RETRY_DELAY=1000
CORS_ORIGIN=*
LOG_LEVEL=info
DB_POOL_MAX=10
DB_POOL_MIN=2
DB_IDLE_TIMEOUT=30000
DB_CONNECTION_TIMEOUT=2000
DB_RETRY_ATTEMPTS=5
DB_RETRY_DELAY=3000

=== HOW TO GET VALUES ===

1. OPENAI_API_KEY:
   - Go to https://platform.openai.com/api-keys
   - Create new secret key
   - Copy and save (won't show again)

2. OPIK_API_KEY:
   - Go to https://www.comet.com/opik
   - Sign up/login
   - Go to Settings â†’ API Keys
   - Create new key

3. JWT_SECRET:
   - Generate random string:
     node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
   - Or use: https://generate-secret.vercel.app/32

4. Database credentials:
   - Automatically shown after creating PostgreSQL service
   - Use INTERNAL connection URL (not external)
   - Format: postgres://user:pass@host:port/db

5. AI_SERVICE_URL:
   - Copy from AI Python service dashboard
   - Format: https://saverfox-ai-python.onrender.com
   - NO trailing slash
